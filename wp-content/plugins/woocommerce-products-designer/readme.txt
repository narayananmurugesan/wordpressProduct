<h4>Overview</h4>
Woocommerce Products Designer (WPD) is a WooCommerce extension which allows your customers to create their perfect logo, t-shirts, business cards, prints, labels, decals, stickers, vinyls, lettering, mugs and any product which needs to be customized online prior to order.

You can use this extension as a: 
<ul>
<li>Woocommerce t-shirt designer</li>
<li>Mug custom editor for WordPress</li>
<li>Woocommerce Phone case customizer</li>
<li>WordPress Stickers, canvas or decals designer</li>
<li>Online business cards, postcards and other print products customizer for WordPress</li>
<li><a href="https://designersuiteforwp.com/products/lettering-and-signs-designer/?utm_source=wordpress.org&utm_medium=cpc&utm_campaign=Lettering%20Signs%20Designer" target="_blank">Woocommerce Lettering Designer</a></li>
<li>Woocommerce Decals Designer</li>
<li>Woocommerce Signs Designer</li>
<li>...and much more.</li>
</ul>


Woocommerce Products Designer is built using WordPress best practices both on the front and the back end. This results in an efficient, robust and intuitive plugin. It’s works with any theme, including the default WordPress themes and the last Woocommerce version.

<div class="block-content"><h4>Minimum Requirements</h4>

<p><strong>This section describes how to install the plugin and get it working</strong></p>

<h4>Automatic installation (easiest way)</h4>

<p>To do an automatic install of Woocommerce Product Designer, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.</p>

<p>In the search field type "Woocommerce Product Designer"� and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".</p>

<h4>Manual installation</h4>

<p><strong>Uploading in WordPress Dashboard</strong></p>

<ol>
<li>Download <code>woocommerce-product-designer.zip</code></li>
<li>Navigate to the 'Add New' in the plugins dashboard</li>
<li>Navigate to the 'Upload' area</li>
<li>Select <code>woocommerce-product-designer.zip</code> from your computer</li>
<li>Click 'Install Now'</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p><strong>Using FTP</strong></p>

<ol>
<li>Download <code>woocommerce-product-designer.zip</code></li>
<li>Extract the <code>woocommerce-product-designer</code> directory to your computer</li>
<li>Upload the <code>woocommerce-product-designer</code> directory to the <code>/wp-content/plugins/</code> directory</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p>The WordPress codex contains <a href="http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation" rel="nofollow">instructions on how to install a WordPress plugin</a>.</p>

<h4>Updating</h4>

<p>You can use automatic update to update the plugin safely.</p></div>
